Detailed Description

template<typename T>
class lfds::SPSCQueue< T >

A lock free, low overhead, single producer/consumer queue. No locks of any kind are used, so no locking/context switch overhead. This is achieved using atomic variables.

Note
Storage for queue is dynamically allocated at construction
Template Parameters
TType of objects contained by the queue
