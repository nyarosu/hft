Detailed Description

template<typename T>
class utils::MemPool< T >

Dynamically allocated memory pool. A large amount of memory is allocated up front, during construction. Then, you can request chunks of memory from the pool, and this will be much faster as the memory has already been allocated.

Template Parameters
TType of object held in pool, must have a default constructor
